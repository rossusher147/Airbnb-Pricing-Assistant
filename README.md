# Airbnb Pricing Assistant

ML-powered Airbnb pricing prediction app with data cleaning, model training, API, and frontend.

As a graduate software developer currently working for CGI. I am working on this project in my spare time to build my skills in the deployemnt of ML models.

## Structure

This repository is organized around an end-to-end machine learning workflow:

- `data/` stores datasets as they move from raw input to model-ready output.
- `notebooks/` is for exploration, experiments, and training workflows in Jupyter.
- `src/APA/` contains the production Python code for the API, data processing, training, and inference.
- `frontend/` is reserved for the user-facing web application.
- `models/` stores trained model artifacts and related files.
- `tests/` contains automated test coverage for the codebase.
- `scripts/` is for repeatable command-line utilities and project tasks.
- `docs/` is for longer-form documentation beyond this overview.

Each major folder contains its own `README.md` with more specific guidance.
